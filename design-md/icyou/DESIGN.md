# ICYOU Design System

**Name:** ICYOU  
**Tagline:** Freedom in every thread

## Visual Style

### Silhouettes
- Oversized, flowy shapes
- Dropped shoulders
- Wide-leg pants
- Extended sleeves
- Relaxed fit throughout

### Details
- Minimal visible stitching
- Hidden buttons
- Subtle pleats
- Raw edges on select pieces
- Monochrome palette with accent colors

### Textures
- Cotton blends
- Linen
- Lightweight nylon
- Soft knits

---

## UI Components

### Navigation
- Minimal top bar
- Logo left-aligned
- Links: Shop, About, Contact
- Clean, no background until scroll

### Buttons
- Primary: Solid charcoal, white text
- Secondary: Outlined, charcoal border
- Hover: Subtle background shift
- Border-radius: 0 (sharp edges) or 4px (subtle)

### Cards (Product)
- Large image area
- Minimal text below
- Price + name only
- No shadows, subtle border

### Layout
- Grid: 2-4 columns (responsive)
- Generous whitespace
- Asymmetric elements allowed

---

## Animations

- Fade-in on scroll: 0.3s ease-out
- Image hover: subtle scale (1.02)
- Link underline: expand from center
- Page transitions: smooth fade

---

## Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
